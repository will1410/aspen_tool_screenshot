About this project

The genesis for this project is simple. I wanted to add collapsible screenshots to basic pages in Aspen Discovery. Since Aspen uses Bootstrap, all of the Aspen tools for creating accordions and buttons, and other Bootstrap styled content is built in, so adding that content is just a matter of writing the HTML for basic pages to include that content. This makes collapsible screenshots possible.

The problem is that creating all of that HTML is kind of a pain. I first used templates so that all I had to do was to copy and paste a template into the web builder, but then you have to go in and add the content and change the IDs and some of the classes and that was also a lot of work. I wanted a way to form-fill the basic data and I came up with this form that only asks for three things - Title, URL, and Alt text. Then I added some checkboxes and radio buttons for different styles I wanted to mess around with. The result is the page you see here.

The biggest caveat about the screenshots you create is that they are formatted for Bootstrap 5. If Aspen moves to a different version of Bootstrap, some of the Bootstrap elements that allow the screenshots to collapse may break and the code you add to Aspen may need to be updated.

This entire project is available on Github, so if you'd like to see the source code or if you'd like to download a copy of the project and change it to suit your needs, feel free to do so.
